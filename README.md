# hoth
A Node.js framework based on fastify

## Requirements

Node.js v12 or later.

## Install

```sh
npm install @hoth/cli --global
```

### Quick start

Create a app:

```sh
hoth generate --app-name="myapp" myapp
```

> --app-name option is required

Install dependencies:

```sh
npm install
```

To start the app in dev mode:

```sh
npm run dev
```
