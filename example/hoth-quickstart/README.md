# `hoth-quickstart`

> Fastify Resty Quickstart API application source

### Install and Run

Using NPM:

```sh
$ npm install
$ npm run dev
```

Using Yarn:

```sh
$ yarn
$ yarn dev
```

### Generated routes

Server is listening on http://127.0.0.1:8080
